Here are some examples of how to use the features in poof. They are sorted from simplest to most complex. Each example introduces a new language feature.

All examples are interactive. Edit the source on the left and hit the poof button to see your changes reflected on the right.

Each of the following examples can be compiled with clang and run. The source code and a script to compile them can be found in the poof source repository.
