About

Poof is a small scripting language sitting on top of a C frontend (lexer, preprocessor, parser). Poof understands your C datatypes, and lets you operate on them at `poof` time.

Basic features

  • Iterate over struct, union, and enum members
  • Iterate over sets of structs, unions and enums
  • Filter types and members using builtin predicates
  • Generate new types and code from existing types

Simple use cases

  • Print enum values as strings
  • Implement generic containers & functions
  • Generate discriminated unions

More elaborate use cases

  • Printf/serialize/deserialize arbitrary datatypes
  • Generate code that statically avoids certain classes of bugs
  • Use poof as a library to write custom static analysis code
